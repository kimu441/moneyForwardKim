# 🛡️ 資産形成プロ v6.1

個人の資産形成を支援する Next.js 製の家計管理 Web アプリです。  
PayPay の CSV 明細を取り込み、変動費・固定費・旅行費を分類管理し、週次・月次での収支を可視化します。

## 機能概要

| カテゴリ | 機能 |
|---|---|
| 変動費管理 | PayPay CSV インポート / 現金手動入力 / 週次予算管理 |
| 固定費管理 | 楽天カード確定固定費の登録（毎月25日に自動計上） |
| 旅行費管理 | 変動費・固定費とは独立した旅行専用の支出管理（毎月24日にリセット） |
| 可視化 | 円グラフ（週別支出内訳）/ 棒グラフ（月次推移・週次トレンド）/ ラインチャート（消化率推移） |
| 分析 | 貯蓄率 / カテゴリ別前月比較 / 週次予算消化率の推移（直近8週） |
| 設定 | 月収・週予算・月予算 / 給料日起点日 / 長期貯蓄目標 |
| AI連携 | 家計データをまとめた Claude / ChatGPT 用プロンプトを自動生成 |
| データ保存 | LocalStorage による永続化（6ヶ月超の明細は月次集計に自動圧縮） |

## お金の3分類

このアプリでは支出を以下の3つに完全分離して管理します。

```
月次サイクル（給料日起点）
├── 変動費（週次管理）
│   ├── PayPay CSV 取込
│   ├── 手動入力（現金タグ付き）
│   └── 週予算: 毎週月曜にリセット（デフォルト ¥15,000）
│
├── 確定固定費（楽天カード・月次）
│   ├── ユーザーが設定画面で手動登録
│   └── 毎月25日（給料日）に支出履歴へ自動計上
│
└── 旅行費（月次・独立管理）
    ├── 旅行・帰省などの非日常支出専用
    ├── 変動費・週予算には影響しない
    └── 毎月24日に自動リセット（0円に）
```

### 各分類の影響範囲

| | 今週の残金 | 変動費グラフ | 月次収支計算 |
|---|:---:|:---:|:---:|
| 変動費（PayPay・現金） | ✅ 反映 | ✅ 反映 | ✅ 反映 |
| 確定固定費（楽天） | ❌ 非反映 | ❌ 非反映 | ✅ 反映 |
| 旅行費 | ❌ 非反映 | ❌ 非反映 | ✅ 反映 |

## データの永続化

### 保存される項目

| キー | 内容 | デフォルト値 |
|---|---|---|
| `asset_history` | 変動費の明細履歴（直近6ヶ月） | `[]` |
| `asset_archived_monthly` | 6ヶ月超の明細を月次集計に圧縮したアーカイブ | `[]` |
| `asset_balance` | 今週の残金 | `15000` |
| `asset_total_savings` | 累計貯蓄額 | `0` |
| `asset_target_savings` | 長期貯蓄目標額 | `2000000` |
| `asset_salary_day` | 給料日（起点日） | `25` |
| `asset_weekly_budget` | 週の変動費予算 | `15000` |
| `asset_rakuten_fixed_costs` | 楽天確定固定費リスト | デフォルト2件 |
| `asset_monthly_income` | 月収（手取り） | `0` |
| `asset_variable_budget` | 月の変動費予算 | `120000` |
| `asset_travel_expenses` | 今月の旅行費履歴 | `[]` |
| `asset_last_week_reset` | 前回の週次リセット日時 | `null` |
| `asset_last_rakuten_charge` | 楽天固定費の前回計上月 | `null` |
| `asset_last_travel_reset` | 旅行費の前回リセット月 | `null` |

### 自動リセット・自動計上のルール

```
毎週月曜 00:00
  └ 今週の残金を週予算にリセット（繰り越しを選んだ場合を除く）

毎月25日（給料日）
  └ 楽天確定固定費リストの合計を支出履歴に一括計上

毎月24日
  └ 旅行費履歴を 0 にリセット
```

### LocalStorage 容量管理

- 使用量を設定画面でリアルタイム表示（最大 5MB）
- **直近6ヶ月**の明細はそのまま保持（詳細振り返り可能）
- **6ヶ月超**の明細は月次集計（月・支出合計・貯蓄）に自動圧縮
- 圧縮後も月次グラフで過去のデータを参照可能

### 必要環境

- Node.js 18 以上
- npm / yarn / pnpm

```bash

# 開発サーバー起動
```

```bash
npm run build
npm run start
```

## 使い方ガイド

### 初回設定の手順

1. **⚙️ 設定タブ** を開く
2. `収入・予算` で月収・週予算・月変動費予算を入力
3. `確定固定費` で家賃・保険・サブスクなどを登録
4. `基本設定` で給料日と長期貯蓄目標を設定

### 毎週の使い方

1. PayPay の支出履歴を CSV でダウンロードし、**ホームタブ**の「PayPay明細（CSV）」からインポート
2. 現金払いは「変動費 手動入力」から記録
3. 明細ログのカテゴリが「食費」になっているものは、バッジをクリックして正しいカテゴリに変更
4. 週末に残金を「貯蓄へ回す」または「来週へ繰り越す」で締め処理

### 旅行費の記録

- ホームタブ右側の「✈️ 旅行の支出」から記録
- 今週の残金には影響せず、月次収支の旅行費欄に加算される
- 毎月24日に自動でリセットされる
